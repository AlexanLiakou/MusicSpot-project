# MusicSpot-project
Simple webpage template for a news site. Please check the readme file for more info

-----General Info-----
	
	This project is a simple webpage template used to display an example of a music news site's webpage.
	
-----Technical Info-----
	
	The project is designed using custom HTML/CSS. There is also some custom Javascript used to update the copyright year in the footer. It was mainly designed using CSS grid. This webpage is also fully responsive to all media screens. A separate stylesheet is used for responsive design.
